**Phase:** 01 Foundation (delivers the full site; Narrative + Motion folded in)
**Spec:** `01-UI-SPEC.md` (binding) · **Design authority:** taste-skill + octo "Supercooled Clarity"
**Deliverable:** `docs/index.html` (static, zero-build) → GitHub Pages `/docs`

## Tasks
1. **Scaffold + tokens** — single static `index.html`; CSS custom properties for the full token set (color/type/space/grid); Satoshi + JetBrains Mono via CDN; grain overlay; scroll progress (transform-only). [SITE-01, DSGN-01]
2. **Narrative (8 sections)** — author copy from `main.tex`; asymmetric per-section composition per UI-SPEC §3; ASSUMPTION tags; §04 climax banner + pullquote; honesty foregrounded. [NARR-01..08, SITE-02, HON-01]
3. **Epistemic ledger** — verdict-document table; status left-borders + line-icons + text labels; struck-through refuted row; 6px border on the Assumption-3 row; staggered reveal; mobile→stacked cards. [NARR-06, HON-01]
4. **Signature canvases** — Animation A (liquid→freeze→cymatic nodes, `cos(kr)cos(pθ)`); Animation B (pulse→ice→scatter→bits); shared rAF; offscreen pause. [ANIM-01..04]
5. **A11y/perf** — IntersectionObserver reveals; `prefers-reduced-motion` static fallbacks; transform/opacity only; responsive single-column < 768px. [SITE-03]
6. **Deploy** — enable Pages from `main /docs`; verify live URL serves the site. [SITE-01]

## Guidance / guardrails (from taste-skill)
No Inter, no pure black, one desaturated accent, no neon glows, no emojis, no 3-equal-card rows, no centered hero. Animate transform/opacity only. Grain on a single fixed pointer-events-none layer.

## Done when
All `01-UI-SPEC.md` acceptance criteria pass (see `01-VERIFICATION.md`) and the live URL serves the styled site.
