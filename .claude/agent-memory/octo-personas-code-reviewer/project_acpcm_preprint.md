The repo `/Users/saharbarak/personal/wu` is a single-file LaTeX arXiv preprint (`main.tex`, embedded `thebibliography`, builds with tectonic) titled "Acousto-Cymatic Phase-Change Memory" (ACPCM).

It deliberately mixes established engineering (HAMR, 3D NAND CTF, PCM, acoustic holography, acoustofluidics, DL acoustic inversion, sonification) with contested (EZ/structured water, Orch-OR) and refuted (water memory, Russell octave cosmology) premises.

**Why:** It is an honest "idea paper" hypothesis generator, not a feasibility claim. The single load-bearing conjecture is isolated as "Assumption 3" (`as:commit`); refuted ideas carry no logical load and are labelled motivation-only.

**How to apply:** When reviewing, preserve the speculative premises and honesty guardrails — do NOT recommend removing them or softening the refuted-science labels. Focus critique on LaTeX/typography/reproducibility and internal consistency, not on the scientific validity of the deliberately-fringe motivation. Build engine is tectonic (XeTeX); `amssymb` is intentionally omitted to avoid a `\Bbbk` clash with `newtxmath`.
