Drive the encoding

The pod's data lives in a standing wave f(r,θ) = cos(k·r)·cos(p·θ). The radial mode k cuts concentric tracks; the angular mode p cuts radial sectors. Each track×sector cell is one addressable voxel - one bit. Type a bitstring and write it, freeze to lock the lattice, then read it back.
